This project consisted of building the [Newsweek](https://www.newsweek.com/) webpage using  a self building grid-base framework  (framework.css) to make it responsive. The framework's grid system uses rows and columns to layout content, it‚Äôs built with float technique and uses media queries to gracefully degrade their site as the window size is reduced.
