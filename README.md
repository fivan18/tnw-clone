This project consisted of building the [TNW](https://thenextweb.com/) webpage using  a self building grid-based [framework](css/framework.css) to make it responsive. The framework's grid system uses rows and columns to layout content, it‚Äôs built with float technique and uses media queries to gracefully degrade the page as the window size is reduced. You can see a really good explanation of how the [grid-based framework](https://github.com/fivan18/grid-based-framework) was built [here](https://dev.to/fivan18/how-to-build-a-grid-based-framework-using-the-float-property-2chf).

## Built With

- CSS
- Self building grid-base [framework](css/framework.css)
